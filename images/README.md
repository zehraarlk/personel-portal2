# Görseller klasörü

Bu klasör **tek kaynak**tır. Tüm site görselleri burada durur.

- React tarafında `frontend/public/images/` bu klasöre **bağlıdır** (kopya değil).
- Tarayıcıda yol: `/images/gebze-logo.webp`
- Veritabanındaki yollar genelde `../images/...` şeklindedir; React bunu otomatik düzeltir.

**Yeni görsel eklerken:** Dosyayı buraya koyun. `frontend/src/images` gibi başka yere kopyalamayın.
